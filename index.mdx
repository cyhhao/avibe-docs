# Your AI agent army, commanded from chat

Vibe Remote connects the coding agents already running on your machine to the chat apps you already use.

No laptop. No IDE. Just vibes.

## What Vibe Remote is

Vibe Remote is not an agent framework, not a hosted coding VM, and not a second brain that sits between you and your agent.

It is a local-first remote-control layer:

- Slack, Discord, Telegram, WeChat, and Lark become terminals.
- Claude Code, OpenCode, and Codex run as themselves.
- Each chat thread can become an isolated coding session.
- Your code and agent processes stay on your machine.
- `vibe remote` gives you a secure remote Web UI when localhost is not enough.

## Start quickly

```bash
curl -fsSL https://raw.githubusercontent.com/cyhhao/vibe-remote/master/install.sh | bash && vibe
```

Then follow the browser setup wizard.
