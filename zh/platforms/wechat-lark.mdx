# 微信和 Lark

Vibe Remote 也支持微信和 Lark / 飞书，让 Agent 出现在你真正工作的聊天场景里。

## 微信

微信适合移动优先的个人工作流，以及中国本土日常沟通。

使用应用内设置向导，并跟随平台相关提示。设置完成后，在启用的聊天里发送一条短冒烟测试消息。

## Lark / 飞书

Lark / 飞书适合已经在使用企业聊天、审批和团队协作生态的工作区。

使用设置向导配置凭证，并选择 Vibe Remote 应该响应的 chat 或 channel。

## 共同排错

| 症状 | 检查 |
| --- | --- |
| 消息没有到达 | 平台凭证、app 安装、所选 chat 范围 |
| Agent 没启动 | 后端是否启用、CLI 路径是否有效、工作目录是否存在 |
| 回复出现在错误位置 | Channel 或 chat 路由设置 |
